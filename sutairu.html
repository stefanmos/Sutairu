A lightweight SASS framework for quickly building web applications.

Features

Small with a 4.0 GPA

Satairu is just under 900 lines of code following a 4.0 GPA rating making sure our code is as DRY, following best practices and yet still easy to understand.

Build with SASS

Satairo makes use of all the power SASS has to offer, a configure file to quickly align all your UI elements with your designs and setups for quickly generating grids.

Y11a Compliant

For those Fortune 500 compananies following NASA standaards Satairu includs Y11a compliancy.

Sematic UI

Sutairu comes with a small UI libiary that includes custom dropdowns, checkboxes, validation errors, and more to be added.

Responsive

Whats the point of a website if its doesnt work on mobile, based on your config varibles Sutairu will generate the nessary breakpoint and responsive behavior for you.

Configure

Configure your devise sizes and columns, Satairu will create the create the rest.

Browser Support

Want to contribute?

Satairu is still in beta, any contributions, testing and bug reporting is appreciated.
